# mcp-server-todovu

Model Context Protocol (MCP) server definition for todo.vu.

This repository hosts the `server.json` definition used for registering the todo.vu MCP server in MCP directories and for automated client discovery.

## Server Details
- **Title**: todo.vu
- **Description**: Model Context Protocol server for todo.vu task management and time tracking.
- **Endpoint**: `https://mcp.todo.vu/mcp`
- **Transport**: Streamable HTTP

## Usage
To use this server with an MCP-compatible client, you can reference the definition file in this repository:
`https://raw.githubusercontent.com/mooball/mcp-server-todovu/main/server.json`
